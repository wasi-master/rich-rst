Abbreviation and Acronym Roles
==============================

Inline abbreviation with expansion:

The :abbreviation:`CSS (Cascading Style Sheets)` is used for styling web pages.

Inline acronym with expansion:

The :acronym:`RST (reStructuredText)` format is used for documentation.

Abbreviation without expansion:

The :abbreviation:`HTML` standard.

Acronym without expansion:

The :acronym:`API` endpoint.
